Each Angular expression has exactly one root scope, but any number of child scopes.

Each expression is evaluated against a context (the scope). Such scope is first looked at in the given element where the property is found. If there is no context/scope, then the parent scope of that element is looked at and so on - until the root scope is reached. This is referred as prototypical inheritance in JavaScript.
