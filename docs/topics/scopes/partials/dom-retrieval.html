Scope are attached to the DOM as $scope data property and cam be retrieved for debugging purposed.

The location where the root scope is attached to the DOM is defined by the location of the ng-app directive.

To examine the scope in the debugger:

  1. Inspect the desired element.
  2. The debugger allows you to access the currently selected element in the console as $0 variable.
  3. To retrieve the associated scope in console, execute: angular.element($0).scope()

The scope() function is only available when $compiler.debugInfoEnabled() is true (which is the default)

However, for performance improvement, debugInfoEnabled() should be set to false.
